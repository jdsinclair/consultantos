# ConsultantOS

Your AI-powered consulting co-pilot.

## Features

- **Clients** - Manage consulting engagements with rich context
- **Live Sessions** - Real-time AI assistance during calls
- **Methods** - Your reusable consulting frameworks
- **Personas** - Different AI modes for different contexts
- **Sources** - Documents, websites, repos indexed for RAG

## Tech Stack

- Next.js 14 (App Router)
- Vercel AI SDK
- Drizzle ORM + Neon (PostgreSQL)
- Vercel Blob (file storage)
- Tailwind CSS + Radix UI

## Getting Started

```bash
npm install
cp .env.example .env.local
# Add your API keys to .env.local
npm run dev
```

## Environment Variables

See `.env.example` for required variables.
